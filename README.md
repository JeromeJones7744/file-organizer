# File Organizer (Python)

A simple Python script that automatically organizes files in a folder by type.

## 📌 Features
- Sorts files into categories:
  - Images
  - Videos
  - Documents
  - Music
  - Archives
  - Code
  - Other
- Automatically creates folders if they don’t exist
- Works on any directory you choose

## 🛠️ Technologies Used
- Python 3
- os module
- shutil module
